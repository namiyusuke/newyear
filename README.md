# 無限ループタイリングアニメーション

WebGLを使用して画像を敷き詰め、4方向に無限にループするインタラクティブなアニメーションです。

## 機能

- ✨ WebGLによる高パフォーマンスなレンダリング
- 🖱️ マウスドラッグで自由に移動
- 📱 タッチ操作対応（スマートフォン・タブレット）
- 🖲️ マウスホイール/トラックパッドスクロール対応
- 🔄 4方向に無限ループ
- 💨 慣性スクロール効果

## 使い方

1. プロジェクトフォルダに画像を配置します（例: `sample.jpg`、`pattern.png`など）

2. `app.js` の最後の部分で画像を読み込みます：

```javascript
// この行のコメントを外して、画像ファイル名を指定
loadImage("your-image.jpg");
```

3. ブラウザで `index.html` を開きます

4. マウスやトラックパッドで画面をドラッグしたり、スクロールして楽しめます！

## 画像がない場合

画像を指定しない場合は、グレーのチェッカーパターンが表示されます。

## カスタマイズ

`app.js` の以下の変数を変更することで、動作をカスタマイズできます：

- `scale`: タイルのサイズ（数値が大きいほど、タイルが多く表示されます）
- `friction`: 慣性の減衰率（0.9-0.99の範囲、値が大きいほど長く滑ります）

## ブラウザ対応

WebGLをサポートする主要なブラウザで動作します：

- Chrome
- Firefox
- Safari
- Edge
